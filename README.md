# steamhourboost

1. Run `npm install .` in script directory
2. Install CLI tools: `npm install -g coffee-script forever`
3. Add user accounts: `coffee users.coffee`
4. Start boosting: `forever start -c coffee boost.coffee`

Enjoy.
